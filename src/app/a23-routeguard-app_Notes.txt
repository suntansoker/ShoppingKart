Angular Route Guards
====================
CanActivate      - To check if we can navigate into a route (Employee Detail Route)
CanDeactivate    - To check if we can navigate away from a route (Add Employee Route)
CanLoad          - To check if a lazily loaded module can be loaded (Customers Route)
Resolve          - To complete data retrieval before navigating to a route (Customers Route - Getting data for customer list)

General Steps
--------------
1) Implement Route Guard service
2) Include the service in providers
3) Add the respective route guard property to the route.
